# medical-erp-admin

React admin panel scaffold for Medical ERP.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the app:
   ```bash
   npm start
   ```

## Structure

- `src/components`: shared UI components
- `src/pages`: route pages for dashboard, login, company, category, supplier, customer, medicine, purchase, sales, reports, settings
- `src/services/api.js`: API helper
- `src/App.js`: route definitions
